# 🍈 Jackfruit Acoustic Maturity Classification using Machine Learning

## 📌 Project Overview

This project presents a non-destructive machine learning approach for classifying jackfruit maturity using acoustic signal features extracted from tapping sounds. The goal is to provide a fast, reliable, and cost-effective alternative to traditional manual maturity assessment.

---

## 🎯 Objectives

- Develop a non-destructive maturity classification system.
- Extract acoustic features from jackfruit tapping sounds.
- Compare multiple machine learning algorithms.
- Identify the best-performing classification model.

---

---

## 📊 Model Performance

| Model | Accuracy |
|--------|---------:|
| KNN | 80.39% |
| SVM | 88.24% |
| LightGBM | 90.20% |
| CatBoost | **92.16%** |

CatBoost achieved the highest classification accuracy among all evaluated models.
